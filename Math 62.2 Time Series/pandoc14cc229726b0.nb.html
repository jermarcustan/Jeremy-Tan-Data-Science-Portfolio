data:  z

Since p is greater than alpha, we don’t reject the null hypothesis. H0: p = 0 Since we don’t reject H0, there is no autocorrelation. We don’t proceed with modeling z.

Box.test(ar2.s, type = "Ljung")

    Box-Ljung test

data:  ar2.s
X-squared = 85.312, df = 1, p-value < 2.2e-16

Since p < alpha, we reject H0 and conclude that the data is autocorrelated.
